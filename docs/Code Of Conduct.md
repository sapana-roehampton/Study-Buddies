## Our Pledge
We are committed to creating a respectful, inclusive, accountable, and collaborative working environment throughout the Study Buddies project. As a team, we recognise that professional behaviour, communication, and responsibility are essential to successful software development.

## Expected Behaviour
Participants in this project are expected to:
- Communicate respectfully and professionally.
- Encourage the team and work cooperatively.
- Attend meetings on time and inform in advance if unable to attend.
- Accept constructive feedback and discuss disagreements professionally.
- Use the platform strictly for academic purposes only.

## Unacceptable Behaviour
The following behaviours are not tolerated:
- Harassment, discrimination, or offensive language.
- Sharing personal data without permission.
- Misusing the platform for non-academic or harmful activities.
- Promoting academic dishonesty or cheating.
- Missing multiple meetings without valid explanation.

## Responsibilities
- Team members should follow agreed project rules and deadlines.
- Any disagreements should be politely discussed among team members.
- If necessary, serious issues should be brought to the attention of the course instructor.

## Enforcement and Consequences
If a team member breaches this Code of Conduct, the team will follow these steps:

**1. Discussion:**
The issue will first be discussed in a team meeting and an improvement plan agreed.

**2. Formal Reminder:**
If the issue continues, it will be documented in the meeting minutes with clear expectations and a deadline.

**3. Task Monitoring:**
Responsibilities may be redistributed or tasks closely tracked on the GitHub Kanban board.

If there is no improvement after internal resolution attempts, violations may result in:

- Redistribution of responsibilities within the team.
- Temporary restriction from decision-making responsibilities.
- Removal of access to the project repository.
- If required, escalation to academic staff in line with university procedures.

## Scope
This Code of Conduct applies to:
- GitHub repository interactions
- Project meetings
- Any communication related to the Study Buddies project
